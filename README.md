# Facewell_ai
FaceWell AI Suite is an Android-based AI application designed to detect faces in real time using machine learning and camera technology. The app can automatically capture a face when it is at the perfect distance from the camera, making face detection smarter and easier to use. The main purpose of this project is to demonstrate the integration of artificial intelligence with mobile applications. The application is developed using Kotlin, Android Studio, CameraX, and ML Kit Face Detection technology.

The main features of this application include real-time face detection, automatic face capture, and a simple and user-friendly interface. In the future, more advanced features such as emotion detection, skin analysis, and face authentication may be added. The project is developed by Saim Ayan, Hanzala Akeel, and Ayaz Irshad.
